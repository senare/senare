Features of GOGS

User dashboard, user profile and activity timeline. Access repositories via SSH, HTTP and HTTPS protocols. User, organization and repository management. Repository and organization webhooks, including Slack, Discord and Dingtalk. Repository Git hooks, deploy keys and Git LFS. Repository issues, pull requests, wiki, protected branches and collaboration. Migrate and mirror repositories with wiki from other code hosts. Web editor for quick editing repository files and wiki. Jupyter Notebook and PDF rendering. Authentication via SMTP, LDAP, reverse proxy, GitHub.com and GitHub Enterprise with 2FA. Customize HTML templates, static files and many others. Rich database backend, including PostgreSQL, MySQL, SQLite3 and TiDB. Have localization over 30 languages.
